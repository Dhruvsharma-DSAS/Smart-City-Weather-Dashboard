# 🌦️ Smart City Weather Dashboard

A premium, interactive weather dashboard built with React and Vite. It provides real-time weather analytics, forecasts, and AI-driven "What to Wear" suggestions.

**Live Demo:** [https://smart-city-weather-dashboard-gamma.vercel.app/](https://smart-city-weather-dashboard-gamma.vercel.app/)

---

## ✨ Features

- **Real-time Weather**: Fetches current conditions using the OpenWeatherMap API.
- **Dynamic Advice**: Provides personalized "What to Wear" suggestions based on temperature and conditions.
- **Search History**: Keeps track of recently searched cities for quick access.
- **Theme Toggle**: Switch between sleek Dark and Light modes.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop views.
- **Glassmorphic UI**: High-end aesthetics with smooth transitions and animations.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Modern CSS variables & Grid/Flexbox)
- **API**: OpenWeatherMap API
- **Icons**: Custom dynamic weather icons

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Dhruvsharma-DSAS/Smart-City-Weather-Dashboard.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run in development mode:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
